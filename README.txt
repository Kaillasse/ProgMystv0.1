============================== LANCEMENT DU PROJET PROGMYST
Prérequis logiciels :
Python 3.10.x (testé avec 3.10.11)
Librairies Python nécessaires :
pygame
pillow
numpy
Installation des dépendances :
Ouvrez un terminal dans le dossier du projet et lancez :

pip install pygame pillow numpy

Lancement du jeu :
Dans le terminal, exécutez :

python main.py

(selon votre installation Python)

Sources des assets :
Tuiles, sprites, pnj : https://scrabling.itch.io/ Étoile menu : https://nyknck.itch.io/staranimation
